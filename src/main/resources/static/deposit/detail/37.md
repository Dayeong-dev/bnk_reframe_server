**가입금액:** 0원 통장으로만 가입 가능  
**상품특징:** 사학연금 수급자 전용 압류방지 통장, 연금 입금만 가능하며 연금 입금 실적 시 수수료 면제 혜택 제공  
**예금과목:** 자유저축예금 (1인 1계좌)  
**가입방법:** 영업점 창구  
**필요서류:** 실명확인증표  
**이자지급방식:** 일일 잔액 기준, 분기별 결산 시 원금에 가산 지급

**압류방지:** 법원의 압류명령에도 압류 불가, 연금 수급권 보호  
**유의사항:** 연금 이외 입금 불가, 연금 수급권 보호를 위한 별도 통장 사용 필요
